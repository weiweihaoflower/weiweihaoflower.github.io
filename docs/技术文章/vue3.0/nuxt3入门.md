# Nuxt.js 是什么？
Nuxt.js 是一个基于 Vue.js 的通用应用框架。
通过对客户端/服务端基础架构的抽象组织，Nuxt.js 主要关注的是应用的 UI 渲染。
我们的目标是创建一个灵活的应用框架，你可以基于它初始化新项目的基础结构代码，或者在已有 Node.js 项目中使用 Nuxt.js。

---
# 创建项目
`提示：node.js版本必须在14.6以上`
```javascript
// 1.创建项目
npx create-nuxt-app <project-name>
//2.安装依赖
npm i
```
